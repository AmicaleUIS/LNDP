# Le Nid des Pronos — V0.25.6

## Nouveautés V0.25.6

- Popup d’exploit déclenchée plus rapidement après l’enregistrement d’un prono.
- Rafraîchissement des pronos visibles avant le contrôle des nouveaux exploits.
- Contrôle des exploits avant puis après le rechargement de la vue, pour éviter l’effet “ça arrive plus tard”.
- File d’attente des popups sécurisée : pas de doublon dans la queue, fermeture protégée, apparition du suivant plus stable.
- Animation d’arrivée plus courte et plus nette pour le popup magique.

## Base de données

Aucun nouveau patch SQL nécessaire pour la V0.25.6.
