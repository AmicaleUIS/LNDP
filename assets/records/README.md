# Mini-records du nid — images

Les mini-records affichent automatiquement une image si elle existe ici :

`assets/records/<record-id>.png`

Format conseillé : PNG ou WebP exporté en PNG, fond transparent, carré `512x512` ou `1024x1024`.
Garde une petite marge autour du dessin pour éviter que l’image touche le bord de la capsule.

## Fichiers attendus

- `record-points.png` — Grand-duc du classement
- `record-average.png` — Moyenne de velours
- `record-exact.png` — Aimant à scores exacts
- `record-results.png` — Collectionneur de victoires
- `record-diffs.png` — Compas du nid
- `record-qualified.png` — Gardien des qualifiés
- `record-day.png` — Journée stratosphérique
- `record-exact-streak.png` — Série laser
- `record-result-streak.png` — Vol sans trou d’air
- `record-predictions.png` — Greffier du grimoire
- `record-zero.png` — Casserole dorée
- `record-zero-streak.png` — Tunnel de brouillard

Si un fichier manque, l’application affiche automatiquement l’icône de secours.
