# Badges d’exploit — Le Nid des Pronos

Les fichiers PNG de ce dossier sont utilisés automatiquement par l’application via le nom de l’exploit :

`assets/badges/<badge-id>.png`

Donc pour un exploit dont l’identifiant est `egg-hatched`, l’image attendue est :

`assets/badges/egg-hatched.png`
