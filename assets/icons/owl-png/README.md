# Icônes chouette PNG

Icônes découpées depuis les planches générées. Elles sont prêtes pour une intégration progressive dans l’interface.

Les planches complètes sont dans `sheets/`.

## Image du bouton retour haut

Le bouton flottant de retour rapide vers le haut utilise automatiquement :

`assets/icons/owl-png/retour-haut-chouette.png`

Format conseillé : PNG carré `512x512`, fond transparent.
Si ce fichier n’existe pas encore, l’application revient automatiquement sur `accueil.png`.
